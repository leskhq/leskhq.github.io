There are multiple ways to acquire a copy of LESK. You can download a ZIP archive, clone the project and finally fork your own repository then clone it.

Option 1: Download

Go to the main repository page at LESK and click on the "Download" button.

Option 2: Clone

Clone this repository on your machine using the command:

git clone https://github.com/leskhq/laravel-enterprise-starter-kit.git lesk

Option 3: Fork & Clone

Finally, the recommended method, specially if you plan on contributing would be to first fork this repository then clone your own repository. Follow the GitHub guide Fork a Repo to learn how to fork an existing repository. Then use a command similar to this to clone your own repository:

git clone https://github.com/YOUR-NAME/laravel-enterprise-starter-kit.git lesk
